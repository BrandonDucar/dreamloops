# ZOL Example

ZOL is the first external reference integration for the persistent-agent starter kit. The example demonstrates how an existing music-curator agent can add general daily planning, durable memory, relationship continuity, project tracking, research, recovery, bounded self-improvement, and optional Warper Keeper assignments without replacing its existing persona or Farcaster implementation.

Recommended composition:

```text
persistent-agent-base
+ daily-life
+ knowledge-and-research
+ communication-and-approval
+ self-improvement
+ ZOL profile
+ optional warper-keeper
```

The example profile contains references and preferences only. It contains no keys, signer material, wallet authority, or deployment authority.
