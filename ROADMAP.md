# Roadmap

## Stable Runtime

- Publish signed schema bundles and package artifacts.
- Add optional SQLite and Postgres state adapters behind the existing store interface.
- Add receipt signatures and external verifier interfaces.
- Add Capsule dependency resolution without arbitrary installation.

## Agent Ecosystem

- Complete the ZOL reference integration.
- Dogfood the same release with DreamNet agents.
- Add reference adapters for MCP, local files, GitHub, calendars, and messaging systems.
- Publish certification fixtures for persistent-agent recovery and memory behavior.

## Public Innovation

- Run Capsule and DreamLoop RFCs in GitHub Discussions.
- Promote proven Grok Labs work into stable extensions.
- Publish reproducible Proof Drops for releases and external integrations.
- Maintain beginner examples and good-first contribution issues.
