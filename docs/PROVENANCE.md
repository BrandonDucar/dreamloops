# Provenance

DreamLoops was extracted into a fresh public repository so private Git history, infrastructure configuration, and unrelated operational artifacts would not be published accidentally.

## Initial Sources

- DreamNet reusable runtime, schemas, DreamLoop protocol, templates, Capsule examples, Warper Keeper client, and architecture papers were reviewed at commit `01fc370f712c81a835afc0d4a97ebf99ed1ec030`.
- Grok Labs Capsule, DreamLoop, Fish Swarm, Genealogist, DNA Engine, consensus, telemetry, proof-ledger, and Persistent Dream materials were reviewed at commit `8e8f65fb3b815f8cc02b87d311d28c9af751688f`.
- The generalized persistent-agent Capsules and DreamLoops were generated in this repository from the public contract in `scripts/generate-starter-kit.mjs`.

Generated operational logs, private connector payloads, environment files, local memory, deployment state, and secret-bearing files were intentionally excluded.

## Promotion Rule

Source provenance is not production certification. Labs and library materials become stable only after schema reconciliation, tests, security review, and a tagged release.
