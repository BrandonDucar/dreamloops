# Grok Labs

Grok Labs is the public experimental lane for Capsule evolution, Fish Swarm coordination, Genealogist lineage, DNA, consensus, telemetry, proof-ledger, and Persistent Dream research.

Files in this directory are exploratory. They may use symbolic or speculative language. Loading them grants no permissions, proves no historical claim, and does not activate any runtime. Stable ideas graduate through an RFC, canonical schemas, tests, receipts, and a tagged release.
