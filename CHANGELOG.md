# Changelog

## 0.1.0 - 2026-07-14

- Published Capsule and DreamLoop specifications and schemas.
- Added dependency-free validation, canonical hashing, Capsule composition, durable file state, bounded execution, and receipts.
- Added CLI initialization, validation, sealing, listing, and dry-run commands.
- Added seven generalized persistent-agent Capsules and eighteen daily-operation DreamLoops.
- Added the optional Warper Keeper client and contract tests.
- Added the OKF exporter.
- Imported the complete reusable DreamNet DreamLoop library and Capsule examples.
- Added Grok Labs with Capsule, Fish Swarm, Genealogist, DNA, consensus, proof-ledger, telemetry, and Persistent Dream experiments.
- Added the public ZOL integration profile and full installation prompt.
