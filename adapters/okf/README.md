# Open Knowledge Format Bridge

DreamLoops describes bounded action. OKF describes portable knowledge. The bridge exports human-readable DreamLoops as discoverable OKF-style playbooks without changing the authoritative source manifests.

```bash
npx dreamloops-okf --source ./library/dreamnet --output ./exports/okf
```

The bridge is local-only. It does not publish, deploy, modify memory systems, or call external services.
